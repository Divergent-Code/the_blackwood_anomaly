# 🌲 The Blackwood Anomaly: Project Outline

Status: 🚧 In Development

## 📖 Concept

The Blackwood Anomaly is an upcoming terminal-based survival horror RPG powered by an AI Dungeon Master. It is being built as an extension of the Module 1 Game Glitch Investigator project.

Instead of a Streamlit number guesser, this project hollows out the base logic to create a persistent game loop where a player's Terminal inputs are processed by an AI that strictly follows retrieved horror mechanics.

## 🎯 Core Features Planned

Multi-Source RAG: The AI will retrieve context from local Markdown files (world_lore.md and combat_mechanics.md) to enforce pacing, non-Euclidean geometry, and survival rules.

Terminal Game Loop: A persistent CLI loop tracking hidden Health and Stress states.

Reliability Testing: Automated pytest scripts to verify input validation and rule retrieval.
